E-Commerce Sales Analysis Project

Tools Used:
SQL, Excel

Description:
This project analyzes e-commerce sales data using SQL queries
and visualizes insights through an Excel dashboard.

Key Analysis:
- Sales by Category
- Monthly Sales Trends
- Payment Mode Analysis
- Top Performing States
- Customer Segmentation

Dashboard:
Excel dashboard showing sales performance and trends.

Skills Demonstrated:
SQL joins, aggregations, window functions, data analysis, dashboard creation.
